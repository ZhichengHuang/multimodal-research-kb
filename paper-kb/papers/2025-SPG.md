## 一句话总结

SPG 通过对 dLLM 的不可计算 log-likelihood 构建"三明治"上下界（正 advantage 最大化 ELBO 下界，负 advantage 最小化 EUBO 上界），解决了现有 dLLM RL 方法因单侧近似导致的 policy gradient 偏差问题，在 GSM8K/MATH500/Countdown/Sudoku 上显著超越 D1/WD1/UniGRPO。

## 核心 Insight

dLLM 的 RL 对齐面临 log-likelihood 不可解析的挑战。现有方法统一使用 ELBO（下界）作为代理，但对负 advantage 样本最小化 ELBO 并不能保证降低真实 log-likelihood——这是一个根本性的数学缺陷。SPG 的核心洞察是：**根据 advantage 符号选择性地使用上/下界**——正 advantage 用 ELBO（最大化下界 ≤ 最大化真值），负 advantage 用 EUBO（最小化上界 ≥ 最小化真值），从而构成真实目标函数的有效下界。这种"三明治"策略将数学上不正确的近似替换为有理论保证的优化目标。

## 方法细节

### 问题形式化
- 目标: 最大化 J(θ) = E_{x~π_θ}[R(c,x)]
- Policy gradient: ∇J(θ) = E[R(c,x) · ∇log π_θ(x|c)]
- 核心挑战: log π_θ(x|c) 对 dLLM 不可解析

### 现有方法的缺陷
- 现有方法（D1/UniGRPO/WD1）用 ELBO 替代 log π_θ
- ELBO ≤ log π_θ，因此:
  - 正 advantage: 最大化 ELBO ≤ 最大化 log π_θ ✓（有效下界）
  - 负 advantage: 最小化 ELBO ≤ 最小化 log π_θ ✗（最小化下界不保证最小化真值）
- 这导致模型无法有效从负反馈中学习

### SPG 算法
1. **正 advantage 样本**: 最大化 LELBO（标准 ELBO 下界）
2. **负 advantage 样本**: 最小化 LEUBO（新推导的上界）
3. 组合: J_SPG(θ) = Σ [1_{A≥0} · A · LELBO + 1_{A<0} · A · L̃_EUBO]
4. 因为 LELBO ≤ log π_θ ≤ LEUBO，所以 J_SPG(θ) ≤ J_group(θ)，构成有效下界

### EUBO 推导
- 基于 Rényi Variational Bound（Rényi 1961）
- 对 β ≥ 1: L̃_EUBO = (1/β) Σ_i log E_{t,z_t}[w(t) · 1(z_{t,i}=m) · π_θ^β(x_i|z_t)]
- β 控制上界的松紧度——β 越接近 1 越紧
- 关键结构差异: EUBO 中 log 在期望**外面**（vs ELBO log 在期望里面），Monte Carlo 估计有偏

### Mixture 策略
- 纯 EUBO Monte Carlo 估计方差大、不稳定
- L̃_Mix = ω · L̃_EUBO + (1-ω) · LELBO，ω ∈ [0,1]
- Proposition 1 证明: 最优混合系数 ω* 的梯度方差严格小于纯 EUBO 或纯 ELBO
- 实践中固定 ω=0.5，β 从 {1.0, 1.5, 2.0} 中选择

### Block-wise Masking 策略
- 动机: LLaDA 推理时用 semi-AR block-wise unmasking，但训练时用 random masking → 训练-推理分布不匹配
- 方法: 序列分成若干 block，随机选一个 block 做 masking（前面 block 保持 clean，后面 block 全 mask）
- 额外在 prompt 和 clean block 上以 p_mask=0.15 轻微扰动增强稳定性
- 效果极大: Countdown 上 block-wise vs random masking 提升 +23.9%（SPG w/ EUBO）

## Building Blocks（可复用组件）

### Block 1: Sandwiched Likelihood Estimation（三明治似然度估计）
- **做法**: 根据 advantage 符号选择性使用 ELBO（正）或 EUBO（负）估计 log-likelihood，替代统一使用 ELBO
- **机制 (WHY it works)**: ELBO ≤ log π ≤ EUBO，对正 advantage 最大化下界有效，对负 advantage 最小化上界有效，组合后 J_SPG ≤ J_true 构成有效优化代理。消除了"最小化 ELBO 不等价于最小化 log-likelihood"的数学缺陷
- **适用条件**: 任何使用 GRPO 风格 advantage-weighted policy gradient 的 dLLM RL 场景；需要 masked diffusion 框架（EUBO 推导依赖 mask-predict 结构）
- **什么时候会 break**: (1) β 选择不当导致 EUBO 过松（β 太大）或 Monte Carlo 估计方差过大（β=1 但采样不足）；(2) 当 advantage 分布极度不平衡（几乎全正或全负）时退化为单侧估计；(3) 对非 masked diffusion 的 dLLM（如 flow matching）不直接适用
- **可组合方向**: 与 EBPO shrinkage baseline（改进 advantage 估计）正交可叠加——SPG 改善 likelihood 估计精度，EBPO 改善 advantage 估计精度；可与 complementary masking 的 likelihood 估计对比/融合

### Block 2: Evidence Upper Bound via Rényi Divergence（基于 Rényi 散度的证据上界）
- **做法**: 利用 Rényi Variational Bound 推导 EUBO: L̃_EUBO = (1/β) Σ_i log E[w(t)·1(m)·π_θ^β(x|z_t)]。与 ELBO 的关键区别是 log 在期望外面，且概率被提升至 β 次方
- **机制 (WHY it works)**: Rényi divergence 的 β-power 提升效应——对高置信 token 权重增加，对低置信 token 权重降低。这实现了 confidence-aware 惩罚：模型更有"把握"预测的 token 受到更强的负惩罚信号
- **适用条件**: 需要 β ≥ 1；需要足够的 Monte Carlo 样本估计 log E[·]（因 log 在期望外）；masked diffusion 框架
- **什么时候会 break**: (1) Monte Carlo 样本不足时 log E[·] 的估计偏差随 β 增大而加剧（Jensen gap）；(2) 在高维/长序列中 per-position 独立估计可能累积误差；(3) 对连续扩散模型需要重新推导
- **可组合方向**: 可将 EUBO 的 confidence-aware 权重思想引入其他 dLLM 训练目标（如 SFT weighted loss）；β 的自适应调整可与训练动态联动

### Block 3: Block-wise Masking for Monte Carlo Estimation（块状掩码 MC 估计）
- **做法**: Monte Carlo 估计 ELBO/EUBO 时，按推理时的 semi-AR block 结构生成 masked 样本（而非 random masking），对齐训练-推理分布
- **机制 (WHY it works)**: LLaDA 等 dLLM 推理时用 semi-AR block-wise unmasking，实际遇到的中间状态是结构化的（前面 clean 后面 mask），random masking 生成的中间状态与此严重不匹配。Block-wise masking 消除这种分布偏移，使 MC 估计更准确
- **适用条件**: 使用 semi-AR block-wise 解码策略的 dLLM；block size 需与推理时一致（实验中均为 32）
- **什么时候会 break**: (1) 推理策略变化（如切换到 random unmasking）时，但 Figure 6 显示即使推理策略改变，block-wise masking 训练的模型仍泛化良好；(2) 非常短的序列使得 block 划分无意义
- **可组合方向**: 可推广到任何需要 MC 估计 dLLM 目标的场景（SFT、DPO 等），不限于 RL；与 SDAR-VL 的块状扩散训练策略有思想共鸣

### Block 4: ELBO-EUBO Mixture for Negative Advantage（混合下界上界惩罚）
- **做法**: 对负 advantage 样本使用 L̃_Mix = ω·L̃_EUBO + (1-ω)·LELBO 替代纯 EUBO，平衡精度和稳定性
- **机制 (WHY it works)**: Proposition 1 证明最优 ω* 的梯度方差严格小于纯 EUBO (ω=1) 或纯 ELBO (ω=0)——因为两者的 noise 不完全正相关，凸组合可在不牺牲信号强度的前提下降方差。EUBO 提供正确方向但方差大，ELBO 稳定但方向偏，混合取两者优点
- **适用条件**: 负 advantage 样本数量足够；β 和 ω 需联合调优（实验中 ω=0.5 即可）
- **什么时候会 break**: (1) 当 EUBO 估计极度不稳定（采样严重不足）时混合也无法挽救；(2) ω 的最优值理论上随训练变化，固定 ω 可能在训练后期不优
- **可组合方向**: 自适应 ω 调度（根据梯度方差在线调整）；per-position 的 ω 选择（不同 token 位置的估计难度不同）

## Anti-patterns / 已知失败模式
- 纯 EUBO（ω=1）训练不稳定，收敛慢——虽然理论上更精确但 Monte Carlo 估计方差太大
- EUBO 的 log(x) ≤ x-1 松弛虽然无偏但性能更差（Table 10）——松弛后 EUBO 与真实 log-likelihood 差距过大
- 纯 ELBO 用于负 advantage 时训练快速 plateau（Figure 4）——最小化下界无法有效降低真实 log-likelihood
- β < 1 的中间值在 Sudoku 上不稳定——既不是严格 ELBO 也不是严格 EUBO，缺乏理论保证
- 去掉负 advantage loss（SPG wo/ neg）性能大幅下降——说明从负反馈学习对 RL 至关重要

## 实验关键发现
- **显著提升**: GSM8K +3.6%, MATH500 +2.6%, Countdown +18.4%, Sudoku +27.0%（对比前 SOTA）
- **Block-wise masking 效果巨大**: Countdown 上 block-wise vs random 提升 +23.9%（SPG w/ EUBO），说明训练-推理分布对齐是 dLLM RL 的关键瓶颈
- **Mixture 优于纯 EUBO**: SPG w/ Mixture 在 4 个任务中全面领先，验证方差降低的实际价值
- **推理策略鲁棒性**: 尽管训练时用 semi-AR block=32 confidence-based，切换到其他推理策略（不同 block size、random unmasking、full sequence）后仍大幅领先所有 baseline
- **β 和 ω 的影响**: β 接近 1.0 通常更好（更紧的上界）；ω 最优在 0.3-0.7 之间，呈倒 U 形
- **所有实验基于 LLaDA-8B-Instruct + LoRA r=128**: 未在其他 dLLM 骨干上验证
- **generation length 256, denoising steps 128**: 固定设置下的结果

## 问题定位
### 解决的问题
- **dLLM RL 中 ELBO 用于负 advantage 的数学缺陷**: 首次明确指出最小化 ELBO 下界不保证降低真实 log-likelihood——这是所有现有 dLLM RL 方法（D1/WD1/UniGRPO/VRPO）的共性缺陷。通过引入基于 Rényi divergence 的 EUBO 构成有效三明治下界
- **dLLM RL 的训练-推理分布不匹配**: 通过 block-wise masking 对齐 semi-AR 推理的结构化中间状态，揭示了训练时 random masking 与推理时 block-wise unmasking 之间的严重分布偏移（Countdown +23.9%）

### 未解决的问题
- **EUBO 在高熵 token 分布下的数值稳定性**:
  - 为什么难: image token NLL>6 意味着 π_θ(x|z_t) ≈ 1/V（V=8192），π_θ^β ≈ V^{-β} 在 β>1 时数值极小，可能导致 log 内期望估计的 underflow。SPG 仅在纯文本推理任务上验证，文本 token 分布熵远低于 image token
  - 潜在思路: per-modality β（text β=1.5, image β=1.0）；对 image token 使用 log-space 计算避免 underflow；或对 image token 退回纯 ELBO（放弃 EUBO 的改进）
- **EUBO Monte Carlo 估计的 Jensen Gap 量化**:
  - 为什么难: L̃_EUBO 中 log 在期望外面，MC 估计 log(1/m · Σf(x_τ)) 有 Jensen bias，偏差随 f 的方差增大而增大，且依赖于 β、序列长度、mask ratio 等多个因素的交互
  - 潜在思路: 推导 Jensen gap 关于 β 和 MC 样本数 m 的上界；使用 control variates 降低 MC 估计方差；second-order correction（log E[X] ≈ E[log X] + Var[X]/(2E[X]²)）
- **SPG 与 LFPO 的系统对比缺失**:
  - 为什么难: 两者从根本不同角度解决同一问题（dLLM RL 的梯度偏差）——SPG 保持 likelihood-based 但提供更紧 bounds，LFPO 完全绕过 likelihood。需要在相同基座、相同任务、相同 reward 下做 Pareto 分析（精度 vs 方差 vs 计算成本）
  - 潜在思路: 在 LLaDA-8B 上用相同训练设置做 SPG vs LFPO head-to-head 对比；分析 token 级 credit assignment 的差异对不同任务类型的影响

### 对问题树的推进
- 推进了:
  - [[problem-tree#[RL-2a] 多模态采样的成本远高于纯文本，如何降低？]]: SPG 提供第六种 dLLM RL likelihood 估计范式——三明治 ELBO+EUBO bounds，与结构化 mask ratio / complementary masking / 无似然度 LFPO / MIS-PO 离散过滤 / shrinkage baseline 并列。Block-wise masking 还揭示了训练-推理分布对齐是一个被严重忽视的效率瓶颈
  - [[problem-tree#[RL-2g] Velocity-based vs Likelihood-based dLLM RL 的系统对比]]: SPG 是 likelihood-based 路线的最新进展——通过更紧的 bounds 显著提升了 likelihood-based 方法的性能上界，使与 LFPO（velocity-based）的对比更有意义
  - [[problem-tree#[RL-3a] 视觉推理 (图表/几何) 的 verifiable reward]]: GSM8K +3.6%, MATH500 +2.6% 的纯文本数学推理提升可直接迁移到多模态数学推理场景
  - [[problem-tree#[RL-3c] 传统 VLM RL vs dLLM RL 的方法论差异]]: SPG 的 EUBO 是 dLLM 专有技术（依赖 masked diffusion 的 ELBO 结构），进一步扩大 dLLM RL 独立方法论体系
- 新增问题:
  - **[RL-2n] Block-wise masking 对齐训练-推理分布的泛化性**: SPG 发现 block-wise masking 在 Countdown 上带来 +23.9% 的巨大提升，暗示现有 dLLM RL 方法的 random masking MC 估计严重偏离推理时实际分布。这一发现是否意味着所有 dLLM 训练（不仅 RL，还有 SFT/DPO）都需要结构化 masking？
  - **[RL-2o] EUBO 在多模态高熵 token 分布下的行为**: SPG 的 β-power 提升在 image token 高熵分布下可能导致数值问题，需要 per-modality 策略

## 个人深度评注
- **真正的增量贡献 vs 工程改进**: SPG 有两个贡献维度，重要性不同:
  - (1) 三明治 ELBO+EUBO 策略是**理论层面的真正创新**——首次指出现有方法对负 advantage 的数学缺陷，并提供有理论保证的修正。EUBO 基于 Rényi divergence 推导，数学上干净优雅
  - (2) Block-wise masking 是**工程层面的关键发现**——效果极大（+23.9%）但本质上是对齐训练-推理分布的常识做法。令人惊讶的是之前没人做过这个显而易见的优化，这反过来说明 dLLM RL 领域仍处于早期，很多基本问题尚未被系统化解决
- **与 KB 中其他方法的位置**: SPG 位于 likelihood-based dLLM RL 的最前沿。KB 中的 dLLM RL 方法现已形成五大范式:
  1. **似然度近似** (UniGRPO: 结构化 mask ratio)
  2. **似然度降方差** (complementary masking w=1)
  3. **advantage 降方差** (EBPO shrinkage baseline)
  4. **无似然度** (LFPO velocity-based)
  5. **似然度 bounds** (SPG: ELBO+EUBO 三明治) ← 新增
  - SPG 与 1/2 是竞争关系（都在改善 likelihood 估计质量），与 3 正交可叠加，与 4 是根本不同路线
- **Block-wise masking 的深层含义**: +23.9% 的效果暗示现有 dLLM RL 的 MC 估计可能有 ~20% 的分布偏移造成的系统性偏差。这一发现对 KB 中所有使用 random masking 做 MC 估计的方法都有警示意义——包括 UniGRPO、complementary masking、d1。可能需要回顾 [P-RL-02] pattern，增加"masking structure"维度
- **对多模态 dLLM RL 的适用性存疑**: SPG 仅在纯文本推理任务（LLaDA-8B + LoRA）上验证。对多模态场景，EUBO 的 β-power 在 image token（NLL>6, 近似均匀分布）上可能有严重数值问题——π_θ^β(x|z_t) ≈ V^{-β}，β=1.5 时为 8192^{-1.5} ≈ 1.3×10^{-6}，积累到整个序列可能导致梯度消失。这限制了 SPG 向统一模型 RL 的直接迁移
